# Linked List

Interactive C++ console program for practicing singly linked list operations.

## Features

- Insert values
- Search for a value
- Count list length
- Update the first node
- Delete a node by value
- Display the list
- Sort values in ascending order

## Build and Run

```bash
g++ -std=c++17 linkedlist_transversal.cpp -o linked-list
./linked-list
```

On Windows PowerShell:

```powershell
g++ -std=c++17 linkedlist_transversal.cpp -o linked-list.exe
.\linked-list.exe
```

## Notes

The program is menu-driven and expects input from the terminal.
